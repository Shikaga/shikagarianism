Welcome to GitHub Pages.

Many people around the world are interested in adopting a religion, but can't find one which can fit into the their post-enlightenment lifestyles.

Enter, Shikagarianism. Shikagarianism is a new religion which can be accepted by people of any background and expertise. Even if this is your first religion you will find Shikagarianism simple to pick up and add to your daily life.

Religion Compatability

In addition, you can adopt Shikagarianism along side your current religion. Shikagarianism is compatible with most religious beliefs including:

  • Monotheism
  • Polytheism
  • Atheism
  • Monism

Browser Comptability

One of the advantages of converting to Shikagarianism is that, as a modern religion, it also comes with a useful JavaScript API, which is compatible with most HTML5 browsers, including:

  • Internet Explorer 9 and 10
  • Chrome
  • Firefox
  • Safari
  • Opera

Extensability

Shikagarianism is also very open to being able to be extended to meet your lifestyle, and if you think you have something to add, then feel free to create your own denomination. And you will automatically updated whenever the core belief set of Shikagarianism is updated.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
